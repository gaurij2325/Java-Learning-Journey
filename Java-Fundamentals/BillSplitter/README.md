# 💸 Bill Splitter (Java)

## 📌 Description
This is a simple console-based Bill Splitter application built using Java.  
It calculates how much each person should contribute when a total bill is shared among multiple people.

The project focuses on basic arithmetic operations, type casting, and user input handling.

---

## 🚀 Features
- 💰 Calculates per-person share of total bill
- 👥 Supports multiple users
- 🔢 Displays both:
  - Actual value (decimal)
  - Rounded-off value (integer)
- ⚡ Simple and user-friendly input system

---

## 🧠 Concepts Used
- Arithmetic operations
- Type casting (double → int)
- User input using Scanner
- Variables and basic calculations
