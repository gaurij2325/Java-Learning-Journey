# 🏦 Mini ATM System (Java)

## 📌 Description
This is a console-based Mini ATM System built using Java.  
It simulates basic banking operations such as checking balance, depositing money, and withdrawing funds through a menu-driven interface.

The project focuses on implementing core programming concepts and building structured logic for real-world scenarios.

---

## 🚀 Features
- 🔐 Menu-driven interface
- 💰 Check account balance
- ➕ Deposit money
- ➖ Withdraw money with validation
- ⚠️ Prevents withdrawal beyond available balance
- 🔁 Continuous operation using loops

---

## 🧠 Concepts Used
- Switch-case statements
- Loops (do-while)
- Conditional statements (if-else)
- Variables and state management
- User input handling using Scanner

## ▶️ How to Run

1. Compile the program: javac miniATM.java
2. Run the program: java miniATM
